# IdFiT 지적∙발달 장애인을 위한 건강관리 서비스💪🏻
### 🔎 IdFiT 소개
#### 지적∙발달 장애인을 위한 ‘개인 운동 추천과 건강관리’ 및 ‘보호자 모니터링’을 제공하는 서비스

<br/>

### ⚙️ 주요 기능
- **보호자 모니터링 서비스<br/>**
